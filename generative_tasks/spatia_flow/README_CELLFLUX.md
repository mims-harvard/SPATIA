# CellFlux

This directory contains the CellFlux codebase, adapted for SPATIA's cell image generation pipeline. For SPATIA-specific usage instructions, see [SPATIA_CELLFLUX_README.md](SPATIA_CELLFLUX_README.md).

For the original CellFlux paper and method details, see [the CellFlux repository](https://github.com/snap-stanford/CellFlux).

## Pretrained Checkpoints

Download CellFlux pretrained checkpoints from [HuggingFace](https://huggingface.co/Perturbation/CellFlux) and place them in the `pretrained_checkpoints/` directory.
